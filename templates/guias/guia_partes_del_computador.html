🖥️ Guía de las Partes del Computador

CPU (Procesador)

Es el cerebro del computador. Ejecuta programas, realiza cálculos y coordina el funcionamiento de todos los componentes.

RAM

Almacena temporalmente la información que utilizan los programas mientras el computador está encendido.

GPU

Procesa los gráficos y genera las imágenes que ves en el monitor.

Motherboard

Es la placa principal donde se conectan todos los componentes del computador.

SSD

Dispositivo de almacenamiento muy rápido donde se guardan el sistema operativo, programas y archivos.

Disco Duro (HDD)

Almacena información de forma permanente utilizando discos magnéticos.

Fuente de Poder (PSU)

Convierte la energía eléctrica para alimentar correctamente todos los componentes del computador.

Monitor

Muestra toda la información y las imágenes generadas por el computador.

Teclado

Permite escribir texto, números y controlar diferentes funciones del computador.

Mouse

Permite mover el cursor, hacer clic, seleccionar elementos e interactuar con la interfaz.
